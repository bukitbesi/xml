## Promotion rules

- `pending`: discovery record only; do not expose phone or hours.
- `partial`: address and source verified; any unverified fields remain blank.
- `verified`: address, phone, hours, source and `lastVerified` all checked.
- Never copy business hours from snippets without confirming the first-party source.
- Store the exact source URL for each branch, not only the Pos Malaysia homepage, when available.
